About this Recipes App
* this will be made for people who wants to look for recipes from all over globe and that too with nutritinal data in it for better healthy eating
* this app will be seo friendly and shareable across social media by users
* this app will also use full stack with modern stack to do so
* this app will use shadui on top of tailwind for css
* orm will be used for mongodb
* will be using design pallets for contents and page layouts
* will be keeping a track of this project using notion and will be producing codes in sprints
* there will be influences from designs inspo found in various places such as dribble
